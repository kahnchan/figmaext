# i18n UI 改进说明

## 📋 改进内容

根据用户反馈，对 i18n 功能的 UI 进行了全面优化，使其与 PRD 和 Tracker 功能保持一致的设计风格和交互体验。

## ✅ 主要改进

### 1. **UI 结构重新设计**

**改进前：**
- 使用简单的 `view` 布局
- 缺少层级感
- 信息展示不够清晰

**改进后：**
- 采用与 PRD 相同的 `content` + `card` 布局
- 层级分明，信息组织清晰
- 视觉风格统一

### 2. **选中图层提示**

**改进前：**
- 没有明确显示当前选中的 Frame
- 用户不清楚正在操作哪个图层

**改进后：**
- 顶部 **Card 1** 显示"Selected Frame(s)"
- 单个 Frame：显示名称和文本节点数量
- 多个 Frames：显示首尾 Frame 名称和总数
- 未选中时：提示用户选择 Frame

```
┌─────────────────────────────────┐
│ Selected Frames (2)             │
│ Login Screen → ... → Success    │
│ 多个设计稿（2 个屏幕）           │
└─────────────────────────────────┘
```

### 3. **简化导出操作**

**移除的功能：**
- ❌ 下载 `.txt` 文件（bulkadd、check）
- ❌ JSON 导出按钮（不常用）
- ❌ CSV 导出按钮（不常用）
- ❌ Check 按钮（不常用）

**保留的核心功能：**
- ✅ 📋 **复制单个 add 命令**（每个 key 卡片上）
- ✅ 📦 **复制 bulkadd 数据**（一键复制，直接粘贴到 Slack）
- ✅ 📊 **创建表格**（在 Figma 画布上创建可编辑表格）

### 4. **优化复制体验**

**改进后：**
- 点击 "📦 复制 bulkadd 数据" 按钮
- 自动复制到剪贴板
- 弹出提示框，显示：
  - ✓ 复制成功
  - 共多少个 keys
  - 使用步骤说明

```
✓ 已复制 bulkadd 数据到剪贴板

共 5 个 keys

使用方法：
1. 在 Slack 输入 @Loka-AI bulkadd
2. 选择项目 v5
3. 粘贴数据
```

### 5. **视觉风格统一**

**Key Card 样式：**
- 使用 CSS 变量（`var(--primary)`、`var(--border)`）
- 支持深色/浅色主题
- 选中状态：蓝色边框 + 淡蓝背景
- Hover 效果：平滑过渡

**按钮样式：**
- 与 PRD 功能保持一致
- 主要操作：`btnPrimary`（蓝色）
- 次要操作：`btn`（灰色）
- 图标按钮：`iconBtn`

## 🎯 用户操作流程

### 完整流程（改进后）

1. **选择 Frame**
   - 在 Figma 中选择一个或多个 Frame
   - 插件顶部 Card 显示选中的 Frame 信息

2. **生成 Keys**
   - 输入项目名称（默认：v5）
   - 点击 "🔤 生成 i18n Keys"
   - 等待 AI 处理

3. **审核 Keys**
   - 查看生成的 keys 列表
   - 勾选/取消勾选需要的 keys
   - 点击 ✏️ 编辑 key 或 value
   - 点击 🗑️ 删除不需要的 key

4. **复制数据**
   - 点击 "📦 复制 bulkadd 数据"
   - 数据自动复制到剪贴板
   - 查看弹窗提示的使用步骤

5. **在 Slack 中使用**
   - 打开 Slack
   - 输入 `@Loka-AI bulkadd`
   - 选择项目 `v5`
   - 粘贴数据
   - 提交

### 单个 Key 操作

- 点击 key 卡片上的 📋 按钮
- 自动复制 add 命令：`@Loka-AI add v5 button_submit Submit`
- 直接在 Slack 粘贴即可

## 🎨 视觉改进

### 颜色变量

使用 CSS 变量适配深色/浅色主题：
- `var(--bg)` - 背景色
- `var(--surface)` - 卡片背景
- `var(--border)` - 边框颜色
- `var(--text)` - 文本颜色
- `var(--subtext)` - 次要文本
- `var(--primary)` - 主色调（蓝色）

### Key Card 样式

**选中状态：**
- 边框：`var(--primary)` 蓝色
- 背景：`rgba(91, 140, 255, 0.05)` 淡蓝色

**未选中状态：**
- 边框：`var(--border)` 灰色
- 背景：`var(--surface)` 卡片背景色

## 📝 代码改进

### UI 组件

**改进前：**
```tsx
<div className="view">
  <div className="controls">...</div>
  <div className="result">
    <div className="stats">...</div>
    <div className="keys-list">...</div>
    <div className="export-section">...</div>
  </div>
</div>
```

**改进后：**
```tsx
<div className="content">
  <div className="card">
    {/* Selected Frames Info */}
  </div>
  <div className="card">
    {/* i18n Keys & Actions */}
  </div>
</div>
```

### 复制操作

**改进前：**
```tsx
onClick={() => postMessage({ 
  type: 'EXPORT_I18N', 
  format: 'bulkadd', 
  projectName 
})}
// 触发下载 .txt 文件
```

**改进后：**
```tsx
onClick={() => {
  const bulkaddData = selectedKeys
    .map(k => `${k.key} | ${k.value}`)
    .join('\n');
  // 直接复制到剪贴板
  document.execCommand('copy');
  // 显示使用提示
  alert(`✓ 已复制 ... 使用方法：...`);
}}
```
